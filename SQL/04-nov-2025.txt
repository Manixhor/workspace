-> What are view functions stores procedures as current databases ?
1.Views-> A view is a virtual table that displays data from one or more tables using a SQL query. 
2.Functions -> A function is a set of SQL statments that performs a operation and returns a single value or table.
3.Stored Procedures -> A stored procedure is a precomplied set of SQL statements stored in the database.

-> what is difference b/w timestamp and date 
| Feature            | **DATE**                          | **TIMESTAMP**                           |
| ------------------ | --------------------------------- | --------------------------------------- |
| **Stores**         | Year, Month, Day                  | Year, Month, Day, Hour, Minute, Second  |
| **Time Zone**      | ❌ No                              | ✅ Optional (depends on DB)              |
| **Typical Format** | `YYYY-MM-DD`                      | `YYYY-MM-DD HH:MI:SS`                   |
| **Storage Size**   | Smaller                           | Larger                                  |
| **Use Case**       | Birthdays, holidays, static dates | Log times, transactions, event tracking |
