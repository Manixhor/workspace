#what is git roll back ?
	Git roll back means going back to the previous state of our code. 
	->  git checkout --fastapi 
	-> git checkout --main.py 
	-> git reset HEAD fastapi 
	-> git reset --hard HEAD~1
	-> git revert <commit-hash>
